1. Introdução

O Praxis CRM é um sistema de gestão de relacionamento com clientes desenvolvido para equipes comerciais. Ele permite gerenciar leads, acompanhar oportunidades de vendas e monitorar a performance do time.

Principais funcionalidades:
  • Cadastro e gestão de clientes (leads)
  • Pipeline de vendas com etapas personalizadas
  • Registro de interações com clientes
  • Metas comerciais e acompanhamento de performance
  • Dashboards para gestão

2. Perfis de Usuário

O sistema possui três perfis de acesso, cada um com permissões específicas:

Perfil Descrição Principais Acessos
VENDEDOR Responsável pela prospecção e fechamento de vendas Seus próprios clientes, oportunidades e metas
GESTOR Acompanha a performance do time comercial Visualização de todos os dados (somente leitura)
ADMIN Administrador do sistema Acesso total, incluindo configurações

3. Acesso ao Sistema

3.1 Login

Para acessar o sistema:

  1. Abra o navegador e acesse o endereço do sistema
  2. Na tela de login, informe seu usuário e senha
  3. Clique em Entrar
Importante: Caso esqueça sua senha, entre em contato com o administrador do sistema para redefinição.

3.2 Menu de Navegação

Após o login, você verá a barra de navegação no topo da tela com os seguintes itens:

Menu Descrição Disponível para

Oportunidades Lista e cadastro de oportunidades Todos
Interações Registro de contatos com clientes Todos
Minhas Pendências Follow-ups atrasados ou para hoje Vendedor
Minha Meta Acompanhamento da meta pessoal Vendedor

4. Guia do Vendedor

4.1 Cadastrando um Cliente

  1. Acesse Clientes no menu
  2. Clique em Novo Cliente
  3. Preencha os dados obrigatórios:
    • Nome: Nome completo ou razão social
    • Tipo: B2B (empresa) ou B2C (pessoa física)
  4. Preencha os dados opcionais (telefone, e-mail, contato principal)
  5. Clique em Salvar
Dica: O cliente será criado com status Provisório. Isso permite cadastro rápido sem burocracia. O CNPJ/CPF será exigido apenas quando o admin for ativar o cliente.

4.2 Cadastrando uma Oportunidade

  1. Acesse Oportunidades no menu
  2. Clique em Nova Oportunidade
  3. Preencha:
    • Título: Nome descritivo da oportunidade
    • Cliente: Selecione o cliente relacionado
    • Valor estimado: Valor previsto da venda
    • Descrição: Detalhes adicionais (opcional)
  4. Clique em Salvar

4.3 Avançando no Pipeline

Cada oportunidade passa por etapas até o fechamento:

Prospecção → Qualificação → Proposta → Negociação → Fechamento

Para avançar uma oportunidade:

  1. Acesse a oportunidade clicando nela na lista
  2. Na tela de detalhes, clique em Avançar Etapa
  3. A oportunidade passará para a próxima etapa automaticamente
Atenção: Oportunidades marcadas como Perdida ou já em Fechamento não podem avançar.

4.4 Registrando Interações

Registre todos os contatos realizados com o cliente:

  1. Acesse Interações no menu
  2. Clique em Nova Interação
  3. Preencha:
    • Oportunidade: Selecione a oportunidade relacionada
    • Tipo: Ligação, E-mail, Reunião, WhatsApp ou Visita
    • Descrição: Resumo do que foi tratado
  4. Clique em Salvar

4.5 Acompanhando sua Meta

  1. Acesse Minha Meta no menu
  2. Visualize:
    • Meta: Valor definido para o mês
    • Realizado: Vendas fechadas no mês
    • Pipeline: Oportunidades em andamento
    • Status: Situação da meta (OK, Atenção ou Risco)

6. Guia do Administrador

O administrador possui acesso total ao sistema, incluindo o painel administrativo do Django.

6.1 Acessando o Django Admin

  1. Clique no seu nome no canto superior direito
  2. Selecione Admin
  3. Ou acesse diretamente: /admin/

6.2 Gerenciando Usuários

  1. No Admin, acesse Usuários
  2. Para criar um novo usuário:
    • Clique em Adicionar usuário
    • Defina usuário e senha
    • Na próxima tela, defina o Perfil (Vendedor, Gestor ou Admin)
    • Marque É da equipe se o usuário precisar acessar o Admin

6.3 Ativando Clientes

Clientes criados por vendedores ficam com status Provisório até serem aprovados:

  1. No Admin, acesse Clientes
  2. Localize o cliente provisório
  3. Verifique se o CNPJ/CPF está preenchido
  4. Selecione o cliente e use a ação Ativar clientes selecionados
Regra: O CNPJ/CPF é obrigatório para ativar um cliente. Caso não esteja preenchido, a ativação será bloqueada.

6.4 Cadastrando Metas

  1. No Admin, acesse Metas Comerciais
  2. Clique em Adicionar meta comercial
  3. Preencha:
    • Vendedor: Selecione o vendedor
    • Mês: Mês da meta
    • Ano: Ano da meta
    • Valor da Meta: Valor monetário
  4. Clique em Salvar
Regra: Cada vendedor pode ter apenas UMA meta por mês/ano. Se tentar criar uma duplicada, o sistema bloqueará.

7. Pipeline de Vendas

O pipeline é o caminho que uma oportunidade percorre desde o primeiro contato até o fechamento da venda.

7.1 Etapas do Pipeline

Prospecção

Primeiro contato com o potencial cliente. Ainda não há demanda validada.

Exemplos: Lead frio, indicação, primeiro contato via WhatsApp.

Qualificação

O vendedor validou que o lead tem potencial real de compra.

Exemplos: Confirmação de volume, identificação do decisor, pedido de cotação.

Proposta

Uma proposta comercial foi apresentada ao cliente.

Exemplos: Orçamento enviado, proposta formal por e-mail.

Negociação

Cliente demonstrou intenção de fechar, mas negocia condições.

Exemplos: Ajuste de preço, negociação de prazos.

Fechamento

Venda concluída com sucesso.

Exemplos: Pedido confirmado, contrato assinado.

Perdida

Venda não realizada.

Exemplos: Cliente desistiu, concorrente venceu.

8. Metas Comerciais

8.1 Como funcionam as Metas

As metas são definidas mensalmente pelo administrador para cada vendedor.

8.2 Cálculo do Realizado

Realizado = Soma dos valores das oportunidades com status Fechamento no mês/ano.

8.3 Cálculo do Pipeline

Pipeline = Soma dos valores das oportunidades em andamento (excluindo Fechamento e Perdida).

8.4 Status da Meta

Status Condição Significado
OK Pipeline ≥ 1,5× a meta Boa cobertura de pipeline
ATENÇÃO Pipeline ≥ meta Cobertura mínima
RISCO Pipeline < meta Necessita ação urgente
Por que 1,5×? Nem todas as oportunidades se convertem em venda. Ter um pipeline de pelo menos 1,5× a meta garante uma margem de segurança para atingir o objetivo.

9. Follow-up e Disciplina Comercial

O follow-up é uma das práticas mais importantes para o sucesso em vendas. No Praxis CRM, o sistema ajuda você a manter a disciplina comercial, garantindo que nenhuma oportunidade fique esquecida.

Princípio: O sistema lembra, você age. O Praxis não envia mensagens automáticas nem persegue clientes. Ele apenas organiza suas pendências para que você nunca esqueça de dar o próximo passo.

9.1 O que é Follow-up

Follow-up é o acompanhamento sistemático de cada oportunidade de venda. Significa definir qual será a próxima ação e quando ela deve acontecer.

Exemplos de próximas ações:

  • Ligar para confirmar recebimento da proposta
  • Enviar e-mail com condições especiais
  • Agendar reunião de fechamento
  • Verificar se o cliente tomou decisão
  • Retornar após período de férias do cliente

9.2 Campos de Follow-up

Cada oportunidade possui dois campos para controle de follow-up:

Campo Descrição Exemplo
Próxima Ação O que você precisa fazer "Ligar para saber decisão sobre proposta"
Data do Follow-up Quando você deve agir 15/02/2026
Importante: Apenas o VENDEDOR responsável pela oportunidade pode editar esses campos. Gestores e Admins apenas visualizam.

9.3 Como Definir o Follow-up

Para definir ou editar o follow-up de uma oportunidade:

  1. Acesse a oportunidade na lista ou busca
  2. Na tela de detalhes, clique em Editar Follow-up
  3. Preencha:
    • Próxima ação: Descreva brevemente o que precisa fazer
    • Data do follow-up: Quando você deve realizar a ação
  4. Clique em Salvar
Dica: Sempre que finalizar um contato com o cliente, já defina a próxima ação. Isso mantém a oportunidade sempre em movimento.

9.4 Lista de Pendências do Vendedor

O vendedor tem acesso a uma lista especial que mostra todas as suas pendências de follow-up:

  1. Acesse Minhas Pendências no menu
  2. Visualize todas as oportunidades que precisam de ação

A lista mostra:

Coluna Descrição
Cliente Nome do cliente da oportunidade
Etapa Em qual fase do pipeline está
Próxima Ação O que você definiu que precisa fazer
Data Follow-up Quando deveria ter sido feito
Dias em Atraso Quantos dias passou da data
Ordenação: A lista mostra primeiro as oportunidades mais atrasadas, para que você priorize o que está pendente há mais tempo.

9.5 Indicadores Visuais de Status

O sistema usa cores para indicar a situação de cada follow-up:

Status Condição Significado
Em dia Data do follow-up ainda não chegou Tudo sob controle
Hoje Data do follow-up é hoje Hora de agir!
Atrasado Data do follow-up já passou Ação urgente necessária

9.6 Visão do Gestor

O GESTOR e o ADMIN têm acesso a uma visão especial para monitorar a disciplina comercial do time:

  1. Acesse Gestão → Oportunidades sem Follow-up
  2. Visualize oportunidades que precisam de atenção

Esta tela mostra:

  • Oportunidades sem data de follow-up definida
  • Oportunidades paradas há muitos dias sem movimentação

Informações exibidas:

Coluna Descrição
Cliente Nome do cliente
Vendedor Responsável pela oportunidade
Etapa Fase atual no pipeline
Dias sem Movimentação Há quantos dias a oportunidade não é atualizada
Atenção Gestores: Esta visão é apenas para monitoramento. O gestor não pode editar o follow-up das oportunidades. Converse com o vendedor responsável quando identificar oportunidades esquecidas.

9.7 Boas Práticas de Follow-up

Regra de ouro: Toda oportunidade ativa deve ter uma próxima ação definida com data.

Para o Vendedor

  1. Defina a próxima ação imediatamente — Ao terminar uma ligação ou reunião, já registre qual será o próximo passo
  2. Seja específico — Em vez de "retornar", escreva "ligar para confirmar se recebeu proposta"
  3. Use datas realistas — Não adiante demais para não esquecer, nem coloque para amanhã sem necessidade
  4. Consulte suas pendências diariamente — Comece o dia verificando a lista de pendências
  5. Atualize após cada contato — Não deixe informações desatualizadas no sistema

Para o Gestor

  1. Monitore semanalmente — Verifique oportunidades sem follow-up pelo menos uma vez por semana
  2. Converse, não persiga — Use as informações para orientar o vendedor, não para cobrança excessiva
  3. Identifique padrões — Se um vendedor sempre esquece follow-ups, pode ser sinal de sobrecarga ou falta de método

10. Lembretes e Alertas

O Praxis CRM possui um sistema de lembretes para ajudar você a manter a disciplina comercial. O sistema avisa, mas você decide quando e como agir.

Princípio: O sistema envia lembretes simples e objetivos. Não há spam, não há automações invasivas. Você recebe apenas o necessário para não esquecer suas pendências.

10.1 Lembrete Visual no Dashboard

Ao acessar o sistema, o VENDEDOR verá um aviso no dashboard quando houver oportunidades com follow-up agendado para o dia:

Lembrete: Você possui X oportunidades com follow-up para hoje. Ver pendências

Este aviso:

  • Aparece apenas quando há follow-ups para o dia atual
  • Mostra a quantidade de oportunidades pendentes
  • Possui um link direto para a lista de pendências
  • Não bloqueia nenhuma ação no sistema

10.2 E-mail de Lembrete Diário

O sistema pode enviar um único e-mail por dia para vendedores que possuem follow-ups agendados:

Assunto: Praxis CRM - Lembrete de Follow-up

Olá [Nome],

Você possui X oportunidades com follow-up agendado para hoje.

Acesse o Praxis CRM para visualizar.

Atenciosamente,
Equipe Praxis CRM

Regras do e-mail:

Regra Descrição
Frequência No máximo 1 e-mail por dia
Condição Apenas se houver follow-up para hoje
Conteúdo Mensagem curta, sem detalhes comerciais
Destinatário Apenas o vendedor responsável
Importante: Se não houver follow-ups para o dia, você não receberá nenhum e-mail. O sistema não envia mensagens desnecessárias.

10.3 Alerta para o Gestor

O GESTOR e o ADMIN veem um alerta no dashboard quando há problemas de disciplina comercial:

Alerta de Disciplina Comercial: X oportunidades sem follow-up definido. Y oportunidades paradas há mais de 7 dias. Ver detalhes

Este alerta indica:

  • Oportunidades sem data de follow-up definida
  • Oportunidades que não foram atualizadas há muitos dias
Nota: O gestor não recebe e-mails de lembrete. O alerta é apenas visual, para consulta quando acessar o sistema.

10.4 O que o Sistema NÃO Faz

É importante saber o que o Praxis CRM não faz em relação a lembretes:

O sistema NÃO... Por quê
Envia múltiplos e-mails por dia Evitar sobrecarga de mensagens
Envia WhatsApp ou SMS Manter a comunicação simples e não invasiva
Altera etapas automaticamente Decisões comerciais são suas
Move ou arquiva oportunidades Você mantém o controle total
Envia e-mails para clientes Lembretes são apenas internos

10.5 Boas Práticas

Para o Vendedor

  1. Verifique o dashboard diariamente — O lembrete visual está lá para ajudar
  2. Mantenha follow-up sempre preenchido — Toda oportunidade ativa deve ter uma próxima ação
  3. Configure seu e-mail corretamente — Certifique-se de que seu e-mail no sistema está atualizado
  4. Não ignore os lembretes — Eles existem para ajudar, não para incomodar

Para o Gestor

  1. Monitore o alerta de disciplina — Use-o para identificar oportunidades esquecidas
  2. Oriente a equipe — Se muitas oportunidades estão sem follow-up, pode ser hora de uma conversa
  3. Não use para cobrança excessiva — O objetivo é ajudar, não pressionar

10.6 Quem Recebe o Quê

Tipo de Lembrete Vendedor Gestor Admin

11. Perguntas Frequentes

Por que não consigo ver os clientes de outros vendedores?

Por segurança, cada vendedor vê apenas seus próprios clientes e oportunidades. Gestores e admins podem ver todos os dados.

Por que não consigo ativar um cliente?

Apenas administradores podem ativar clientes. Além disso, o cliente precisa ter o CNPJ/CPF preenchido.

Como marco uma oportunidade como perdida?

Na tela de detalhes da oportunidade, clique em Marcar como Perdida.

Posso editar uma meta depois de criada?

Sim, o administrador pode editar metas no Django Admin.

O que acontece se eu tentar criar uma meta duplicada?

O sistema não permite duas metas para o mesmo vendedor no mesmo mês/ano.

Como sou notificado sobre minha meta?

Acesse a tela Minha Meta regularmente para acompanhar seu progresso. O status (OK, Atenção, Risco) indica sua situação.

O que acontece se eu não definir um follow-up?

A oportunidade ficará visível na tela "Oportunidades sem Follow-up" do gestor. Isso pode indicar falta de disciplina comercial. Sempre defina a próxima ação para manter suas oportunidades em movimento.

Por que não consigo editar o follow-up de uma oportunidade?

Apenas o vendedor responsável pela oportunidade pode editar o follow-up. Gestores e admins podem apenas visualizar.

Como vejo minhas pendências de follow-up?

Acesse Minhas Pendências no menu. A lista mostra todas as oportunidades com follow-up atrasado ou para hoje, ordenadas por prioridade.

Por que não recebi o e-mail de lembrete?

O e-mail só é enviado se você tiver oportunidades com follow-up para o dia atual. Se não houver nenhuma, você não receberá e-mail. Verifique também se seu e-mail está cadastrado corretamente no sistema.

Posso desativar os lembretes por e-mail?

O envio de e-mails é gerenciado pelo administrador do sistema. Converse com ele se precisar ajustar as configurações.

O sistema envia e-mails para meus clientes?

Não. Os lembretes são apenas internos, enviados para você (vendedor). O sistema nunca envia e-mails para clientes.
