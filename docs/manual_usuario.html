1. Introdução

O Praxis CRM é um sistema de gestão de relacionamento com clientes desenvolvido para equipes comerciais. Ele permite gerenciar leads, acompanhar oportunidades de vendas e monitorar a performance do time.

Principais funcionalidades:
  • Cadastro e gestão de clientes (leads)
  • Pipeline de vendas com etapas personalizadas
  • Registro de interações com clientes
  • Metas comerciais e acompanhamento de performance
  • Dashboards para gestão

2. Perfis de Usuário

4. Guia do Vendedor

4.1 Cadastrando um Cliente

  1. Acesse Clientes no menu
  2. Clique em Novo Cliente
  3. Preencha os dados obrigatórios:
    • Nome: Nome completo ou razão social
    • Tipo: B2B (empresa) ou B2C (pessoa física)
  4. Preencha os dados opcionais (telefone, e-mail, contato principal)
  5. Clique em Salvar
Dica: O cliente será criado com status Provisório. Isso permite cadastro rápido sem burocracia. O CNPJ/CPF será exigido apenas quando o admin for ativar o cliente.

4.2 Cadastrando uma Oportunidade

  1. Acesse Oportunidades no menu
  2. Clique em Nova Oportunidade
  3. Preencha:
    • Título: Nome descritivo da oportunidade
    • Cliente: Selecione o cliente relacionado
    • Valor estimado: Valor previsto da venda
    • Descrição: Detalhes adicionais (opcional)
  4. Clique em Salvar

4.3 Avançando no Pipeline

Cada oportunidade passa por etapas até o fechamento:

Prospecção → Qualificação → Proposta → Negociação → Fechamento

Para avançar uma oportunidade:

  1. Acesse a oportunidade clicando nela na lista
  2. Na tela de detalhes, clique em Avançar Etapa
  3. A oportunidade passará para a próxima etapa automaticamente
Atenção: Oportunidades marcadas como Perdida ou já em Fechamento não podem avançar.

4.4 Registrando Interações

Registre todos os contatos realizados com o cliente:

  1. Acesse Interações no menu
  2. Clique em Nova Interação
  3. Preencha:
    • Oportunidade: Selecione a oportunidade relacionada
    • Tipo: Ligação, E-mail, Reunião, WhatsApp ou Visita
    • Descrição: Resumo do que foi tratado
  4. Clique em Salvar

4.5 Acompanhando sua Meta

  1. Acesse Minha Meta no menu
  2. Visualize:
    • Meta: Valor definido para o mês
    • Realizado: Vendas fechadas no mês
    • Pipeline: Oportunidades em andamento
    • Status: Situação da meta (OK, Atenção ou Risco)

6. Guia do Administrador

O administrador possui acesso total ao sistema, incluindo o painel administrativo do Django.

6.1 Acessando o Django Admin

  1. Clique no seu nome no canto superior direito
  2. Selecione Admin
  3. Ou acesse diretamente: /admin/

6.2 Gerenciando Usuários

  1. No Admin, acesse Usuários
  2. Para criar um novo usuário:
    • Clique em Adicionar usuário
    • Defina usuário e senha
    • Na próxima tela, defina o Perfil (Vendedor, Gestor ou Admin)
    • Marque É da equipe se o usuário precisar acessar o Admin

6.3 Ativando Clientes

Clientes criados por vendedores ficam com status Provisório até serem aprovados:

  1. No Admin, acesse Clientes
  2. Localize o cliente provisório
  3. Verifique se o CNPJ/CPF está preenchido
  4. Selecione o cliente e use a ação Ativar clientes selecionados
Regra: O CNPJ/CPF é obrigatório para ativar um cliente. Caso não esteja preenchido, a ativação será bloqueada.

6.4 Cadastrando Metas

  1. No Admin, acesse Metas Comerciais
  2. Clique em Adicionar meta comercial
  3. Preencha:
    • Vendedor: Selecione o vendedor
    • Mês: Mês da meta
    • Ano: Ano da meta
    • Valor da Meta: Valor monetário
  4. Clique em Salvar
Regra: Cada vendedor pode ter apenas UMA meta por mês/ano. Se tentar criar uma duplicada, o sistema bloqueará.

7. Pipeline de Vendas

O pipeline é o caminho que uma oportunidade percorre desde o primeiro contato até o fechamento da venda.

7.1 Etapas do Pipeline

Prospecção

Primeiro contato com o potencial cliente. Ainda não há demanda validada.

Exemplos: Lead frio, indicação, primeiro contato via WhatsApp.

Qualificação

O vendedor validou que o lead tem potencial real de compra.

Exemplos: Confirmação de volume, identificação do decisor, pedido de cotação.

Proposta

Uma proposta comercial foi apresentada ao cliente.

Exemplos: Orçamento enviado, proposta formal por e-mail.

Negociação

Cliente demonstrou intenção de fechar, mas negocia condições.

Exemplos: Ajuste de preço, negociação de prazos.

Fechamento

Venda concluída com sucesso.

Exemplos: Pedido confirmado, contrato assinado.

Perdida

Venda não realizada.

Exemplos: Cliente desistiu, concorrente venceu.

8. Metas Comerciais

8.1 Como funcionam as Metas

As metas são definidas mensalmente pelo administrador para cada vendedor.

8.2 Cálculo do Realizado

Realizado = Soma dos valores das oportunidades com status Fechamento no mês/ano.

8.3 Cálculo do Pipeline

Pipeline = Soma dos valores das oportunidades em andamento (excluindo Fechamento e Perdida).

8.4 Status da Meta

Status Condição Significado
OK Pipeline ≥ 1,5× a meta Boa cobertura de pipeline
ATENÇÃO Pipeline ≥ meta Cobertura mínima
RISCO Pipeline < meta Necessita ação urgente
Por que 1,5×? Nem todas as oportunidades se convertem em venda. Ter um pipeline de pelo menos 1,5× a meta garante uma margem de segurança para atingir o objetivo.

9. Follow-up e Disciplina Comercial

O follow-up é uma das práticas mais importantes para o sucesso em vendas. No Praxis CRM, o sistema ajuda você a manter a disciplina comercial, garantindo que nenhuma oportunidade fique esquecida.

Princípio: O sistema lembra, você age. O Praxis não envia mensagens automáticas nem persegue clientes. Ele apenas organiza suas pendências para que você nunca esqueça de dar o próximo passo.

9.1 O que é Follow-up

Follow-up é o acompanhamento sistemático de cada oportunidade de venda. Significa definir qual será a próxima ação e quando ela deve acontecer.

Exemplos de próximas ações:

  • Ligar para confirmar recebimento da proposta
  • Enviar e-mail com condições especiais
  • Agendar reunião de fechamento
  • Verificar se o cliente tomou decisão
  • Retornar após período de férias do cliente

9.2 Campos de Follow-up

Cada oportunidade possui dois campos para controle de follow-up:

Campo Descrição Exemplo
Próxima Ação O que você precisa fazer "Ligar para saber decisão sobre proposta"
Data do Follow-up Quando você deve agir 15/02/2026
Importante: Apenas o VENDEDOR responsável pela oportunidade pode editar esses campos. Gestores e Admins apenas visualizam.

9.3 Como Definir o Follow-up

Para definir ou editar o follow-up de uma oportunidade:

  1. Acesse a oportunidade na lista ou busca
  2. Na tela de detalhes, clique em Editar Follow-up
  3. Preencha:
    • Próxima ação: Descreva brevemente o que precisa fazer
    • Data do follow-up: Quando você deve realizar a ação
  4. Clique em Salvar
Dica: Sempre que finalizar um contato com o cliente, já defina a próxima ação. Isso mantém a oportunidade sempre em movimento.

9.4 Lista de Pendências do Vendedor

O vendedor tem acesso a uma lista especial que mostra todas as suas pendências de follow-up:

  1. Acesse Minhas Pendências no menu
  2. Visualize todas as oportunidades que precisam de ação

A lista mostra:

Coluna Descrição
Cliente Nome do cliente da oportunidade
Etapa Em qual fase do pipeline está
Próxima Ação O que você definiu que precisa fazer
Data Follow-up Quando deveria ter sido feito
Dias em Atraso Quantos dias passou da data
Ordenação: A lista mostra primeiro as oportunidades mais atrasadas, para que você priorize o que está pendente há mais tempo.

9.5 Indicadores Visuais de Status

O sistema usa cores para indicar a situação de cada follow-up:

Status Condição Significado
Em dia Data do follow-up ainda não chegou Tudo sob controle
Hoje Data do follow-up é hoje Hora de agir!
Atrasado Data do follow-up já passou Ação urgente necessária

9.6 Visão do Gestor

O GESTOR e o ADMIN têm acesso a uma visão especial para monitorar a disciplina comercial do time:

  1. Acesse Gestão → Oportunidades sem Follow-up
  2. Visualize oportunidades que precisam de atenção

Esta tela mostra:

  • Oportunidades sem data de follow-up definida
  • Oportunidades paradas há muitos dias sem movimentação

Informações exibidas:

Coluna Descrição
Cliente Nome do cliente
Vendedor Responsável pela oportunidade
Etapa Fase atual no pipeline
Dias sem Movimentação Há quantos dias a oportunidade não é atualizada
Atenção Gestores: Esta visão é apenas para monitoramento. O gestor não pode editar o follow-up das oportunidades. Converse com o vendedor responsável quando identificar oportunidades esquecidas.

9.7 Boas Práticas de Follow-up

Regra de ouro: Toda oportunidade ativa deve ter uma próxima ação definida com data.

Para o Vendedor

  1. Defina a próxima ação imediatamente — Ao terminar uma ligação ou reunião, já registre qual será o próximo passo
  2. Seja específico — Em vez de "retornar", escreva "ligar para confirmar se recebeu proposta"
  3. Use datas realistas — Não adiante demais para não esquecer, nem coloque para amanhã sem necessidade
  4. Consulte suas pendências diariamente — Comece o dia verificando a lista de pendências
  5. Atualize após cada contato — Não deixe informações desatualizadas no sistema

Para o Gestor

  1. Monitore semanalmente — Verifique oportunidades sem follow-up pelo menos uma vez por semana
  2. Converse, não persiga — Use as informações para orientar o vendedor, não para cobrança excessiva
  3. Identifique padrões — Se um vendedor sempre esquece follow-ups, pode ser sinal de sobrecarga ou falta de método

10. Lembretes e Alertas

O Praxis CRM possui um sistema de lembretes para ajudar você a manter a disciplina comercial. O sistema avisa, mas você decide quando e como agir.

Princípio: O sistema envia lembretes simples e objetivos. Não há spam, não há automações invasivas. Você recebe apenas o necessário para não esquecer suas pendências.

10.1 Lembrete Visual no Dashboard

Ao acessar o sistema, o VENDEDOR verá um aviso no dashboard quando houver oportunidades com follow-up agendado para o dia:

Lembrete: Você possui X oportunidades com follow-up para hoje. Ver pendências

Este aviso:

  • Aparece apenas quando há follow-ups para o dia atual
  • Mostra a quantidade de oportunidades pendentes
  • Possui um link direto para a lista de pendências
  • Não bloqueia nenhuma ação no sistema

10.2 E-mail de Lembrete Diário

O sistema pode enviar um único e-mail por dia para vendedores que possuem follow-ups agendados:

Assunto: Praxis CRM - Lembrete de Follow-up

Olá [Nome],

Você possui X oportunidades com follow-up agendado para hoje.

Acesse o Praxis CRM para visualizar.

Atenciosamente,
Equipe Praxis CRM

Regras do e-mail:

Regra Descrição
Frequência No máximo 1 e-mail por dia
Condição Apenas se houver follow-up para hoje
Conteúdo Mensagem curta, sem detalhes comerciais
Destinatário Apenas o vendedor responsável
Importante: Se não houver follow-ups para o dia, você não receberá nenhum e-mail. O sistema não envia mensagens desnecessárias.

10.3 Alerta para o Gestor

O GESTOR e o ADMIN veem um alerta no dashboard quando há problemas de disciplina comercial:

Alerta de Disciplina Comercial: X oportunidades sem follow-up definido. Y oportunidades paradas há mais de 7 dias. Ver detalhes

Este alerta indica:

  • Oportunidades sem data de follow-up definida
  • Oportunidades que não foram atualizadas há muitos dias
Nota: O gestor não recebe e-mails de lembrete. O alerta é apenas visual, para consulta quando acessar o sistema.

10.4 O que o Sistema NÃO Faz

É importante saber o que o Praxis CRM não faz em relação a lembretes:

O sistema NÃO... Por quê
Envia múltiplos e-mails por dia Evitar sobrecarga de mensagens
Envia WhatsApp ou SMS Manter a comunicação simples e não invasiva
Altera etapas automaticamente Decisões comerciais são suas
Move ou arquiva oportunidades Você mantém o controle total
Envia e-mails para clientes Lembretes são apenas internos

10.5 Boas Práticas

Para o Vendedor

  1. Verifique o dashboard diariamente — O lembrete visual está lá para ajudar
  2. Mantenha follow-up sempre preenchido — Toda oportunidade ativa deve ter uma próxima ação
  3. Configure seu e-mail corretamente — Certifique-se de que seu e-mail no sistema está atualizado
  4. Não ignore os lembretes — Eles existem para ajudar, não para incomodar

Para o Gestor

  1. Monitore o alerta de disciplina — Use-o para identificar oportunidades esquecidas
  2. Oriente a equipe — Se muitas oportunidades estão sem follow-up, pode ser hora de uma conversa
  3. Não use para cobrança excessiva — O objetivo é ajudar, não pressionar

10.6 Quem Recebe o Quê

Tipo de Lembrete Vendedor Gestor Admin
Lembrete visual no dashboard Sim Não Não
E-mail de follow-up Sim Não Não
Alerta de disciplina comercial Não Sim Sim

11. Filtros e Busca

As listas de Oportunidades, Interações e Clientes possuem filtros para ajudar você a localizar informações rapidamente, especialmente quando o volume de registros cresce.

11.1 Filtros de Oportunidades

Na lista de Oportunidades (Oportunidades no menu), você encontra um card de filtros acima da tabela com as seguintes opções:

Filtro Descrição
Cliente Selecione um cliente para ver apenas suas oportunidades
Data inicial Mostra oportunidades criadas a partir desta data
Data final Mostra oportunidades criadas até esta data

Clique em Filtrar para aplicar. Use Limpar para remover todos os filtros e voltar à lista completa.

Dica: Você pode combinar os filtros. Por exemplo, selecione um cliente E um período para ver apenas as oportunidades daquele cliente naquele intervalo de datas.

11.2 Filtros de Interações

Na lista de Interações (Interações no menu), os mesmos filtros estão disponíveis:

  • Cliente: Filtra interações vinculadas a oportunidades do cliente selecionado
  • Data inicial / Data final: Filtra pelo período de criação da interação

11.3 Busca de Clientes

Na lista de Clientes (Clientes no menu), há um campo de busca por nome:

  1. Digite parte do nome do cliente no campo de busca
  2. Clique em Buscar
  3. A lista mostrará apenas os clientes cujo nome contém o texto digitado
Dica: A busca não diferencia maiúsculas de minúsculas. Digitando "silva", você encontrará "Silva", "SILVA" e "Da Silva".

11.4 Filtros e Paginação

Quando você aplica filtros e a lista filtrada possui mais de uma página, os filtros são preservados ao navegar entre as páginas. Você não precisa reaplicá-los a cada página.

11.5 Visibilidade dos Filtros

Perfil Clientes no dropdown
VENDEDOR Apenas seus próprios clientes
GESTOR Todos os clientes
ADMIN Todos os clientes

11.6 Quando Usar Cada Filtro

Filtro por cliente: Use quando quiser ver o histórico completo de um cliente específico — todas as oportunidades ou interações associadas a ele.
Filtro por período: Use quando quiser analisar um intervalo de tempo — por exemplo, oportunidades criadas no último mês ou interações da semana passada.

12. Sustentação da Disciplina (Etapa 5)

A Etapa 5 do Praxis CRM reúne funcionalidades que ajudam a manter a disciplina comercial no dia a dia. O objetivo é simples: garantir que nenhuma oportunidade seja esquecida e que o vendedor tenha ferramentas práticas para organizar seu trabalho.

Princípio da Etapa 5: O sistema ajuda, mas não atrapalha. Os lembretes são discretos, os filtros são opcionais, e nenhuma automação substitui a decisão do vendedor.

12.1 Lembrete Visual

Ao entrar no sistema, o VENDEDOR verá um card discreto no dashboard quando houver oportunidades com follow-up vencido:

Você tem X oportunidades com follow-up vencido. Ver pendências

Características do lembrete:

  • Aparece apenas quando há follow-ups com data passada (vencidos)
  • Mostra a quantidade exata de oportunidades pendentes
  • Possui link direto para a lista de pendências
  • Não bloqueia nenhuma ação — é apenas informativo
  • Desaparece automaticamente quando não há mais pendências

Além do lembrete de vencidos, o sistema também exibe um aviso separado para follow-ups agendados para o dia atual, ajudando o vendedor a priorizar suas ações.

12.2 Lembrete por E-mail

O sistema envia um único e-mail por dia para vendedores que possuem follow-ups agendados:

Característica Descrição
Frequência No máximo 1 e-mail por vendedor por dia
Condição Apenas se houver follow-up agendado para o dia
Conteúdo Mensagem curta informando a quantidade de pendências
Destinatário Apenas o vendedor responsável
Importante: O e-mail não contém detalhes comerciais sensíveis. É apenas um lembrete para que o vendedor acesse o sistema e verifique suas pendências.

12.3 Filtros nas Listas

As listas de Oportunidades e Interações possuem filtros por Cliente e Período (data inicial e data final). A lista de Clientes possui busca por Nome.

Os filtros utilizam parâmetros GET na URL, o que significa que:

  • Os filtros são preservados ao navegar entre páginas
  • Os filtros são mantidos ao voltar de uma ação (editar, visualizar)
  • É possível compartilhar uma URL filtrada com outro usuário

Para mais detalhes sobre cada filtro, consulte a seção 11. Filtros e Busca.

12.4 Objetivo da Etapa

A Etapa 5 não adiciona funcionalidades complexas. Ela conecta recursos existentes para criar um ciclo de sustentação:

Vendedor entra no sistema → Vê lembrete visual de follow-ups vencidos → Acessa lista de pendências → Usa filtros para localizar oportunidades → Atualiza follow-ups → Ciclo se repete no próximo acesso
Resultado esperado: Com o uso diário do lembrete visual, dos e-mails e dos filtros, o vendedor mantém todas as oportunidades em movimento, sem depender de memória ou controles externos.

13. Qualidade de Dados & Leitura Inteligente (Etapa 6)

A Etapa 6 do Praxis CRM foca em melhorar a qualidade dos dados e facilitar a leitura inteligente das informações. O objetivo é ajudar a equipe a tomar decisões mais rápidas e embasadas, sem adicionar burocracia.

Princípio da Etapa 6: Dados de qualidade geram decisões melhores. O sistema guia, não bloqueia. Insights são para aprendizado, não para punição.

13.1 Motivo de Perda

Ao marcar uma oportunidade como perdida, o vendedor agora deve informar o motivo da perda. Isso permite que a equipe analise os padrões e tome ações corretivas.

Como funciona:

  1. Na tela de detalhes da oportunidade, clique em Marcar como perdida
  2. O sistema abrirá um formulário pedindo o motivo
  3. Selecione o motivo principal e confirme

Motivos disponíveis:

Motivo Quando usar
Preço Cliente achou caro ou obteve preço melhor
Prazo Não foi possível atender o prazo do cliente
Concorrência Cliente fechou com outro fornecedor
Desistência Cliente desistiu da compra
Sem retorno Cliente parou de responder
Outro Motivo não listado acima
Por que informar o motivo? Ao acumular dados de perdas, o gestor consegue identificar padrões. Se muitas oportunidades são perdidas por preço, pode ser hora de revisar a política comercial. Se muitas são perdidas por "sem retorno", pode indicar falha no follow-up.

13.2 Aviso de Duplicidade

Ao cadastrar um novo cliente, o sistema verifica se já existe um cliente com nome ou CNPJ/CPF semelhante. Se encontrar, exibe um aviso amarelo listando os possíveis duplicados.

Comportamento:

  • O aviso não bloqueia o cadastro
  • O vendedor pode revisar os duplicados e decidir se deseja continuar
  • Para confirmar, basta clicar em Salvar mesmo assim
  • Se não houver duplicados, o formulário funciona normalmente
Atenção: Este aviso é preventivo. Cadastros duplicados dificultam a análise de dados e podem gerar confusão nas oportunidades. Sempre verifique antes de confirmar.

13.3 Tempo Médio por Etapa

No dashboard do gestor, uma nova tabela mostra quanto tempo, em média, as oportunidades ficam em cada etapa do pipeline.

Coluna Descrição
Etapa Nome da etapa do pipeline
Tempo Médio (dias) Média de dias que as oportunidades ficam na etapa
Registros Quantidade de oportunidades usadas no cálculo
Como interpretar: Se uma etapa tem tempo médio muito alto, pode indicar um gargalo no processo. Por exemplo, se oportunidades ficam paradas na "Proposta" por muitos dias, pode ser necessário agilizar a elaboração de propostas.

13.4 Insights do Dashboard

O dashboard do gestor agora exibe três insights automáticos baseados nos dados do CRM:

Insight O que mostra Para quem
Etapa com mais perdas Em qual etapa do pipeline as oportunidades mais são perdidas GESTOR ADMIN
Cobertura de pipeline Quanto do pipeline cobre a meta total do mês GESTOR ADMIN
Dias médio parado Há quantos dias, em média, as oportunidades abertas estão sem movimentação GESTOR ADMIN
Nota: Os insights são calculados automaticamente a partir dos dados do sistema. Eles aparecem apenas quando há dados suficientes para gerar a informação.

13.5 Objetivo da Etapa

A Etapa 6 existe para melhorar a qualidade da informação no CRM, sem adicionar burocracia. O ciclo é:

Vendedor marca perda com motivo → Gestor vê padrões de perda nos insights → Equipe identifica pontos de melhoria → Ações corretivas são tomadas → Taxas de conversão melhoram
Importante: Os insights são para aprendizado, não para punição. O objetivo é entender onde o processo pode melhorar, não apontar culpados. Use os dados para orientar, treinar e desenvolver a equipe.

14. Perguntas Frequentes

Por que não consigo ver os clientes de outros vendedores?

Por segurança, cada vendedor vê apenas seus próprios clientes e oportunidades. Gestores e admins podem ver todos os dados.

Por que não consigo ativar um cliente?

Apenas administradores podem ativar clientes. Além disso, o cliente precisa ter o CNPJ/CPF preenchido.

Como marco uma oportunidade como perdida?

Na tela de detalhes da oportunidade, clique em Marcar como Perdida. O sistema pedirá o motivo da perda antes de confirmar.

Posso editar uma meta depois de criada?

Sim, o administrador pode editar metas no Django Admin.

O que acontece se eu tentar criar uma meta duplicada?

O sistema não permite duas metas para o mesmo vendedor no mesmo mês/ano.

Como sou notificado sobre minha meta?

Acesse a tela Minha Meta regularmente para acompanhar seu progresso. O status (OK, Atenção, Risco) indica sua situação.

O que acontece se eu não definir um follow-up?

A oportunidade ficará visível na tela "Oportunidades sem Follow-up" do gestor. Isso pode indicar falta de disciplina comercial. Sempre defina a próxima ação para manter suas oportunidades em movimento.

Por que não consigo editar o follow-up de uma oportunidade?

Apenas o vendedor responsável pela oportunidade pode editar o follow-up. Gestores e admins podem apenas visualizar.

Como vejo minhas pendências de follow-up?

Acesse Minhas Pendências no menu. A lista mostra todas as oportunidades com follow-up atrasado ou para hoje, ordenadas por prioridade.

Por que não recebi o e-mail de lembrete?

O e-mail só é enviado se você tiver oportunidades com follow-up para o dia atual. Se não houver nenhuma, você não receberá e-mail. Verifique também se seu e-mail está cadastrado corretamente no sistema.

Posso desativar os lembretes por e-mail?

O envio de e-mails é gerenciado pelo administrador do sistema. Converse com ele se precisar ajustar as configurações.

O sistema envia e-mails para meus clientes?

Não. Os lembretes são apenas internos, enviados para você (vendedor). O sistema nunca envia e-mails para clientes.

12. Módulo de Prospecção

12.1 O que é Prospecção

O módulo de Prospecção é a etapa anterior ao CRM. Enquanto o CRM gerencia clientes já cadastrados e oportunidades de vendas, a Prospecção cuida dos leads — pessoas ou empresas que demonstraram algum interesse, mas que ainda não se tornaram clientes.

Resumo: A Prospecção é o "funil de entrada" do sistema. Nela, você registra contatos iniciais, acompanha tentativas de abordagem e, quando o lead fechar negócio, ele pode ser convertido em cliente no CRM.

Principais benefícios:

  • Centralizar todos os leads em um só lugar
  • Registrar cada tentativa de contato (ligação, WhatsApp, e-mail, visita)
  • Acompanhar automaticamente o status de cada lead
  • Receber lembretes de follow-up para não esquecer nenhum lead
  • Ter visão clara de quantos leads estão em cada fase

12.2 Fluxo de Trabalho

O fluxo de prospecção segue uma sequência simples e intuitiva:

1. Cadastrar Lead ↓ 2. Registrar Contato (ligação, WhatsApp, e-mail, visita) ↓ 3. Sistema atualiza status automaticamente ↓ 4. Agendar próximo contato (follow-up) ↓ 5. Repetir até o lead fechar ou ser descartado
Importante: Você não precisa alterar o status do lead manualmente. O sistema faz isso automaticamente com base no resultado de cada contato registrado.

12.3 Cadastro de Lead

Para cadastrar um novo lead:

  1. No menu superior, clique em Prospecção → Leads
  2. Clique em Novo lead
  3. Preencha os dados:
    • Nome: Nome da pessoa ou contato (obrigatório)
    • Empresa: Empresa do lead (opcional)
    • Telefone: Telefone principal (obrigatório)
    • WhatsApp: Número do WhatsApp (opcional)
    • E-mail: E-mail de contato (opcional)
    • Origem: De onde veio esse lead (obrigatório) — ex: Instagram, Google, Indicação, Loja Física
    • Produto de interesse: O que o lead demonstrou interesse (opcional)
    • Observações: Qualquer informação adicional (opcional)
  4. Clique em Salvar
Dica: Cadastre o lead assim que receber o contato. Quanto mais rápido registrar, menor a chance de esquecer detalhes importantes.

12.4 Registro de Contato

Toda vez que você tentar falar com um lead — com sucesso ou não — registre o contato:

  1. Acesse o lead na lista de leads
  2. Na tela de detalhes, clique em Registrar Contato
  3. Preencha:
    • Tipo de contato: Como foi o contato
    • Resultado: O que aconteceu
    • Observação: Detalhes do que foi conversado (opcional)
    • Data do próximo contato: Quando retornar (opcional)
  4. Clique em Salvar

Tipos de contato disponíveis

Tipo Quando usar
Ligação Chamada telefônica realizada
WhatsApp Mensagem enviada ou conversa via WhatsApp
Presencial Visita à loja, reunião presencial ou encontro
E-mail E-mail enviado ao lead

Resultados possíveis

Resultado Significado Exemplo
Não atendeu Tentou contato, mas não conseguiu falar Chamou e caiu na caixa postal
Pediu retorno Falou com o lead, mas ele pediu para retornar depois "Me liga na semana que vem"
Interessado Lead demonstrou interesse e quer seguir em frente "Me manda a proposta por e-mail"
Sem interesse Lead disse que não quer prosseguir "Não preciso disso no momento"
Fechou Lead confirmou que quer fechar negócio "Vamos fechar, pode emitir o pedido"
Importante: Registre o contato mesmo quando não conseguir falar com o lead. O registro de "Não atendeu" é valioso para saber quantas tentativas foram feitas.

12.5 Status do Lead

O status do lead é atualizado automaticamente pelo sistema cada vez que você registra um contato. Você não precisa alterar o status manualmente.

Status Significado Quando acontece
Novo Lead recém-cadastrado, sem nenhum contato Ao criar o lead
Em contato Já houve tentativa de contato com o lead Após resultado "Não atendeu" ou "Interessado"
Aguardando Lead pediu para retornar depois Após resultado "Pediu retorno"
Convertido Lead fechou negócio e virou cliente Após resultado "Fechou"
Perdido Lead não tem interesse, descartado Após resultado "Sem interesse"
Como funciona: O sistema lê o resultado do contato e atualiza o status do lead automaticamente. Por exemplo, se você registrar um contato com resultado "Pediu retorno", o lead muda para status "Aguardando". Se registrar "Fechou", o lead muda para "Convertido" e a data de conversão é preenchida.

12.6 Follow-ups de Prospecção

O follow-up de prospecção garante que você não esqueça de retornar para nenhum lead. Ao registrar um contato, você pode definir a data do próximo contato.

Como funciona

  1. Ao registrar um contato, preencha o campo Data do próximo contato
  2. O sistema cria automaticamente um lembrete de follow-up para aquela data
  3. Se o lead já tinha um follow-up pendente, o anterior é cancelado e substituído pelo novo
  4. No dia marcado, o follow-up aparece na tela Follow-ups Hoje

Acessando os follow-ups

No menu superior, clique em Prospecção → Follow-ups Hoje. Esta tela mostra:

Coluna Descrição
Lead Nome do lead (clicável para ver detalhes)
Data Data agendada para o retorno
Descrição Resumo automático do que motivou o follow-up
Status do Lead Status atual do lead
Ações Botão para registrar contato diretamente
Atenção: Follow-ups atrasados (data já passou) também aparecem nesta tela, destacados em vermelho. Priorize esses contatos, pois quanto mais tempo passa, menor a chance de conversão.

Badge de notificação

No menu superior, o item Prospecção exibe um contador vermelho quando há follow-ups pendentes (para hoje ou atrasados). Esse contador atualiza automaticamente conforme você registra contatos e completa seus follow-ups.

Status do follow-up

Status Significado
Pendente Ainda precisa ser realizado
Concluído Contato foi realizado
Cancelado Substituído por um follow-up mais recente

12.7 Conversão para CRM

Quando um lead fecha negócio, o fluxo é o seguinte:

  1. Registre um contato com resultado Fechou
  2. O sistema automaticamente:
    • Altera o status do lead para Convertido
    • Registra a data de conversão
  3. Em seguida, cadastre o lead como cliente no CRM (menu Clientes → Novo Cliente)
  4. Crie uma oportunidade para dar continuidade no pipeline de vendas
Resumo do fluxo completo: O lead entra pela Prospecção, é trabalhado com contatos e follow-ups, e quando fecha negócio, é cadastrado como cliente no CRM para acompanhamento da venda pelo pipeline.

12.8 Perguntas Frequentes sobre Prospecção

Preciso alterar o status do lead manualmente?

Não. O status é atualizado automaticamente cada vez que você registra um contato. Basta informar o resultado e o sistema cuida do resto.

Posso registrar um contato mesmo se não consegui falar com o lead?

Sim, e você deve. Selecione o resultado "Não atendeu". Isso mantém o histórico completo de tentativas.

O que acontece se eu esquecer de preencher a data do próximo contato?

Nenhum follow-up será criado. O contato será registrado normalmente, mas você não receberá lembrete de retorno. Recomendamos sempre preencher quando houver necessidade de retorno.

Posso ter mais de um follow-up pendente para o mesmo lead?

Não. Quando você agenda um novo follow-up, o anterior é cancelado automaticamente. Isso evita confusão e garante que apenas o mais recente esteja ativo.

Um lead convertido precisa ser cadastrado como cliente?

Sim. A conversão na Prospecção apenas marca que o lead fechou negócio. Para acompanhar a venda pelo pipeline, cadastre-o como cliente no CRM (menu Clientes → Novo Cliente).

Posso editar um lead após o cadastro?

Sim. Na lista de leads, clique no nome do lead e depois em Editar. Você pode alterar qualquer informação, incluindo o status.

Como sei quantos leads tenho pendentes para hoje?

O menu Prospecção na barra de navegação exibe um contador vermelho com a quantidade de follow-ups pendentes. Clique em Follow-ups Hoje para ver a lista completa.
